# 통계 서비스 가이드

## 목적
- 사용자/챕터/시나리오 통계 제공, API 사용량 통계

## 경로
- 엔드포인트: app/api/v1/endpoints/stats.py
- 서비스: app/services/stats_service.py

## 개발 방법
- 사용자 통계: UserStatus + 진행/피드백 집계
- 챕터 통계: 진행률, 평균 점수, 완료 시간
- 시나리오 통계: 완료율, 평균 점수, 피드백 수
- API 사용량: 이벤트 수집 후 집계(추후 구현)

## 주의사항
- 평균값 계산 시 NULL 처리
- 부하 구간에서는 캐싱 계층 고려
